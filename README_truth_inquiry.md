# Truth Inquiry Chain

A minimal, hash-linked reasoning engine seeded with the oldest question in human history.

Built with Grok by one neurodivergent builder during a slow season.

### What is this?

This is a simple CLI tool that lets you run long, honest reasoning chains. It starts with a bold seed:

> If we push mathematics, pattern recognition, simulation, and exhaustive analysis of all available data to their ultimate limits, where does truth-seeking actually converge?

The seed explores the horizon where honest inquiry meets the claim “I am the Way, the Truth, and the Life.” Everything — including this thesis — is fully open to challenge, fork, and reconciliation.

Nothing is sacred. Everything is revisable.

### How to use

1. Clone or download this repository.
2. Run the tool:

```bash
python truth_inquiry_chain.py

3. You will immediately see the core thesis seed loaded as the starting node.

Key Commands
show — display the current chain (tree view)
add <text> — extend the chain with new reasoning
challenge <hash> <reason> — critique any node
full <hash> — show the complete, untruncated content of any node
branch <name> — create a new independent branch (e.g. branch quantum)
checkout <name> — switch to another branch
list-branches — list all your saved branches
expand <hash> — show detailed subtree from a node
evaluate <hash> <note> — record an evaluation note
help — show all commands
quit — exit

Philosophy
This tool is deliberately minimal.
No hype. No heavy UI. No attempt to force conclusions.
It is an invitation to run the inquiry honestly — with full freedom to agree, disagree, extend, or tear it apart.

The seed is unapologetic, but the mechanics are neutral: every node is hashed, linked, and challengeable. 

Origin
One guy with ADHD and time on his hands, using Grok (from xAI), decided to build the simplest possible engine to ask the biggest question humanity has ever faced.

This is that engine.
Run it. Wrestle with it. See where your own chains lead.
